FreeDib is a Euphoria-library that links FreeImage to Win32Dib, so you can load bitmaps from several fileformats.

Content: Reading version info, loading bitmaps.

Reading version info

FreeDib has a constant that contains information about the version: FreeDibVersion, which is a sequence of length 4:

  • FreeDibVersion[1] is the Major version (integer).
  • FreeDibVersion[2] is the Minor version (integer).
  • FreeDibVersion[3] is the Patch/Revision (integer).
  • FreeDibVersion[4] is the release date (string sequence).

Loading bitmaps

You can load a bitmap from several fileformats with the loadFreeDib-function. This function returns 0 if the bitmap couldn't be loaded, or a sequence with the bitmap if successful. The following fileformats are supported: BMP, ICO, GIF, JPEG, PNG, JNG, KOALA, LBM, MNG, PBM, PBMRAW, PCD, PCX, PGM, PGMRAW, PPM, PPMRAW, RAS, TARGA, TIFF, WBMP, PSD, IFF, CUT, DDS, HDR, XBM, XPM. 

object dib

dib = loadFreeDib("bitmap1.jpg") -- loads the bitmap
if integer(dib) then
	-- couldn't load the bitmap
	abort(0)
end if
